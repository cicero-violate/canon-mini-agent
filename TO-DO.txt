**Math Model**

[
W(s_t, e) \rightarrow s_{t+1} ;; \text{iff} ;; I(e, s_t)=1
]

---

### Variables

* (W): Canonical writer
* (s_t): current state
* (s_{t+1}): next state
* (e): event
* (I(e,s)): invariant validator (0/1)
* (T): total ordered log (tlog)
* (C): control events
* (E_f): effect events

---

### Equations

* (T = [e_1, e_2, \dots, e_n])
* (W: (T, e_{n+1}) \rightarrow T')
* (I(e_{n+1}, s_n)=1 \Rightarrow \text{append})
* (I(e_{n+1}, s_n)=0 \Rightarrow \text{reject})
* (C \rightarrow \text{advances state machine})
* (E_f \rightarrow \text{no control obligation change})

---

### Explanation

**Canonical Writer = Single deterministic gate for all state changes**

Core properties:

* **Single entry point**: every event must pass through (W)
* **Append-only log**: no mutation, only (T \rightarrow T')
* **Invariant enforced**: invalid transitions never enter the system
* **Total order**: strict sequence → no ambiguity, no race conditions
* **Control vs Effect split**:

  * Control events move the system forward
  * Effect events are side outputs, no state transition authority

---

### What it does structurally

* Converts system into:
  [
  \text{State Machine} = f(T)
  ]
* Eliminates:

  * hidden writes
  * parallel inconsistent states
  * invalid transitions
* Enables:

  * replay
  * debugging
  * invariant discovery

---

### Minimal Form

[
\text{if } I(e,s)=1 \Rightarrow T.push(e)
]
[
\text{else } \Rightarrow \text{emit violation + halt path}
]

---

### Explanation (Plain)

It is the **only place in the system allowed to change reality**.

Everything:

* planner
* router
* tools
* LLM

→ must emit events
→ those events must pass the writer

If not, they **do not exist**.

This forces:

* determinism
* correctness
* full observability

\max(I,E,C,A,R,P,S,D,T,K,X,B,L,F) = \text{Good}
